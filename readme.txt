Написать программу, которая из имеющегося массива строк формирует массив из строк, 
длина которых меньше либа равна трем символам. 


Создаем методы, заполнения и печати массивов.
Создаем исходный массив, строки вводим с клавиатуры.
Выводим на печать.

В цикле вводим переменную Индекс, которая будет считать кол-во элементов, длина которых меньше/равна 3.
Создаем новый массив, длина которого равна полученному значения переменной Индекс.

В цикле перебираем, все эелементы первого массива, если они удовлетворяют условию, 
что длина меньше/равна 3 - помещаем их во второй массив.

Печатаем второй массив. 

p.s. напечатала не методом, потому что мой вскод ни в какую не хотел применять PrintArray(array1)
к новому массиву. С циклом обычным все заработало. Он у меня вообще иногда глючит.
